# Playwright MCP makes web scraping fun again

Since I first used [Playwright MCP](https://playwright.dev/) with Claude more than a
year ago, I've been amazed by how fun it has become to scrape websites, extract
front-end elements I like, and more broadly explore the web with LLMs.

More than just fetching URLs, Playwright allows LLMs to click and interact with
webpages, and you can visually track its actions within your browser. A genuinely
useful tool that turns the messy business of web scraping into something almost
playful.
