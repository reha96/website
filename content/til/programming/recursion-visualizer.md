# Debug recursion without asking a statistical parrot

[Recursion Visualizer](https://www.recursionvisualizer.com/) is a mildly interesting
tool that visualizes how your recursive code runs by drawing tree diagrams of function
calls. Useful for debugging complex loops and understanding what your code is actually
doing.

Sure, any LLM today could probably explain your loop to you with far less hassle. But
this is a debugging alternative for those who still don't trust statistical parrots.
