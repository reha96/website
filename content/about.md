## About this site

This site is a static website built with [Next.js 15](https://nextjs.org/), styled with [Tailwind CSS](https://tailwindcss.com/), and deployed to [Vercel](https://vercel.com/). The source code is available on [GitHub](https://github.com/reha96/website).

Blog posts and TIL entries are written in Markdown. The site generates a static export at build time. There is no database and no server-side rendering at runtime.

Color palette drawn from [*A Dictionary of Color Combinations*](https://www.goodreads.com/book/show/140534069-a-dictionary-of-color-combinations) by Sanzo Wada.
